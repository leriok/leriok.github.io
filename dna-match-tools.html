DNA Match Tools

Набор инструментов для анализа ДНК-совпадений внутри Genomelink: от простого списка совпадений до визуализации хромосомных сегментов. Проектировали с нуля — от AI-прототипа до продукта, встроенного в существующий продукт.

Задача

Четыре инструмента — один продукт

Genomelink DNA Match — это набор инструментов для генеалогических исследований. Каждый решает отдельную задачу: DNAMatch управляет списком совпадений и профилями родственников, Network Graph показывает связи между ними в виде графа, Clusters автоматически группирует совпадения по ветвям семьи, Chromosome Canvas визуализирует, какие сегменты хромосом пришли от каких ветвей.

Задача — спроектировать каждый инструмент с нуля, довести UX до продуктового уровня, органично встроить все четыре в единый продукт и подготовить маркетинговые материалы для запуска.

Ключевая задача

Сделать сложные генеалогические инструменты понятными и удобными — для людей, которые ищут родственников, а не изучают генетику.

🧬

DNAMatch

Список совпадений, дедупликация, группировка по линиям, профиль матча.

🕸️

Network Graph

Граф связей между совпадениями — визуально, интерактивно, по ветвям.

🔵

Clusters

Автоматическая кластеризация совпадений по общему ДНК — для определения ветви.

🧩

Chromosome Canvas

Визуализация сегментов на хромосомах — какой участок ДНК от какой ветви.

Процесс

От прототипа до интегрированного продукта

Работа шла в четыре этапа: быстрое прототипирование с помощью AI, доработка UX/UI каждого инструмента, интеграция в единый продукт и подготовка маркетинга.

1 AI-прототипы

Для каждого из четырёх инструментов сделали AI-прототип — быстрый способ проверить концепцию, структуру экранов и основные флоу до детальной проработки. Прототипы позволили выровнять понимание продукта внутри команды и с заказчиком до начала дизайна.

2 UX/UI каждого инструмента

На основе прототипов доработали и докрутили UX/UI каждого инструмента. DNAMatch получил удобный список совпадений с фильтрами по линиям и статусам, систему дедупликации кросс-провайдерных записей и детальный профиль каждого совпадения с хромосомным браузером, сравнением этничности и генетических черт.

Network Graph и Clusters — интерактивные визуализации с чёткой логикой взаимодействия. Chromosome Canvas — инструмент для окраски хромосом по кластерам и линиям с точным отображением сегментов.

Особенность

Инструменты работают в связке: дедупликация в DNAMatch делает кластеризацию точнее, линии из DNAMatch подсвечиваются на графе и на хромосомах. UX должен отражать эту взаимосвязь.

3 Интеграция в DNA Match и навигация

Срастили все четыре инструмента и встроили в существующий DNA Match продукт. Переработали навигацию: список совпадений и инструменты теперь переключаются единым тоглом, между инструментами — единая шапка и общий контекст. Всё, что пользователь настраивает в одном инструменте — линии, кластеры — автоматически доступно во всех остальных.

4 Маркетинговые ассеты

Подготовили маркетинговые материалы для запуска инструментов: визуалы для соцсетей, иллюстрации фич, материалы для email-кампаний. Каждый инструмент — отдельная точка коммуникации с пользователем.

Результат

Единая система инструментов

Четыре инструмента запущены в продукте и продолжают развиваться. Следующий этап — One-to-One инструменты: cM Clarity, Side Assignment Inspector, WATO Workspace, Inferred Segments.

AI-прототипы для всех четырёх инструментов — быстрая верификация концепции до детального дизайна

Network Graph и Clusters — интерактивные визуализации с логикой, связанной с остальными инструментами
