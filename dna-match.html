DNA Match

Отдельное направление внутри Genomelink: поиск родственников через сравнение ДНК. Строили с MVP, постепенно наращивая функционал.

Идея

Найти родственников через ДНК

DNA match — это сервис для поиска родственников внутри Genomelink. Пользователи загружают свои ДНК-данные, система сравнивает их и показывает людей с совпадающим ДНК: процент совпадения, вероятная степень родства, возможность связаться.

Сервис строился итеративно: сначала MVP со списком совпадений, потом — сообщения, загрузка семейного дерева, расширение карточки родственника. Каждая фича встраивалась в уже существующий сценарий.

Ключевая фишка

ДНК как способ найти родственников, которых не знал. Не генеалогическое дерево вручную, а автоматическое совпадение через биологические данные.

Процесс

MVP и постепенное расширение

Работала в команде с двумя дизайнерами и дизайн-лидом. Проанализировали конкурентов, утвердили вайрфреймы, запустили лендинг и приступили к MVP. Каждая новая фича встраивалась в уже существующий сценарий.

1 MVP: первый запуск

Отрисовали первый дизайн сервиса для запуска DNA Match. Создали демо-версию и ждали, пока наберётся база пользователей. Основа — список совпадений с процентом ДНК и вероятной степенью родства.

2 Базовые сценарии для всех типов пользователей

Отрисовали сценарии для всех типов пользователей: экраны онбординга для присоединения к сервису, профиль с заполнением информации, превью профиля, карточку родственника. Покрыли все ключевые состояния.

3 Система сообщений

Когда сервис набрал достаточное количество пользователей, добавили обмен сообщениями между найденными родственниками. Спроектировали функцию запроса контакта и распространили её на все сценарии.

4 Семейное дерево и карточка родственника

Добавили загрузку файла семейного дерева, фамилий и ссылок на дерево пользователя — три новых раздела в редактировании профиля. Для каждого провайдера написали короткую и длинную инструкцию по получению ссылки или файла. Параллельно расширили карточку родственника: добавили лейбл провайдера, количество хромосом, совпадения по фамилии.

Результат

Живой продукт, который продолжает расти

DNA Match постепенно обрастает новыми функциями, каждая из которых встраивается в уже существующий сценарий. В беклоге — улучшение системы фильтров, чат внутри платформы, построение семейного дерева прямо в интерфейсе.

MVP и лендинг с нуля: список совпадений, демо-версия, онбординг для всех типов пользователей

Система сообщений: запрос контакта распределён по всем сценариям и состояниям

Загрузка семейного дерева: три провайдера, короткие и длинные инструкции, спецификации с микроанимациями

Статус

Сервис продолжает развиваться. В беклоге — улучшение фильтров, чат внутри платформы, построение семейного дерева прямо в интерфейсе.
