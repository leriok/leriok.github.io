Taara

Беспроводной интернет через световые лучи. Поддержка маркетинговой команды: техническая документация, гайд по установке, ивент-материалы, соцсети, AI-видео, презентации для инвесторов, темплейты для команды клиента.

Задача

Быстро растущий продукт, сложный скоуп

Taara вышла из X — лаборатории Alphabet (бывшая Google's Moonshot Factory) — как самостоятельная компания. Маркетинг охватывал сразу несколько направлений: техническая документация, материалы для продаж, живые события, digital-каналы. Удерживать качество и консистентность на всех фронтах одновременно — задача с нарастающей сложностью.

Параллельно шли продуктовые запуски, международные ивенты и постоянный поток контента для соцсетей. Каждый формат требовал отдельной адаптации, но все должны были говорить одним визуальным языком.

Контекст

Компания передаёт данные через лучи света — без волокна, через горы и города. Коммуникации должны были объяснять сложную технологию просто и убедительно, в очень разных форматах.

Процесс

Четыре направления параллельно

Работа шла одновременно по нескольким потокам — документация, ивенты, соцсети, видео. У каждого свой темп и свои требования к качеству.

1 Техническая документация

Разрабатывали инструкцию по сборке устройства с нуля. Отсматривали видео реального монтажа и раскладывали процесс на чёткие шаги — так, чтобы техник на объекте мог следовать гайду без дополнительных объяснений. Иллюстратор отрисовывал детали и компоненты по 3D-моделям оборудования. После финальной версии прошли проверку от технической команды клиента — замечаний не было.

Дополнительно разработали печатные версии для вкладки в упаковку оборудования — box-ready print с учётом формата и условий производства.

2 Ивент-дизайн

Дизайн стендов для конференций и выставок: чистые, акцентные решения, которые презентуют продукт и привлекают внимание в насыщенной среде. Большеформатные принты, баннеры, стойки — всё с соблюдением технических требований подрядчиков.

3 Соцсети и контент-шаблоны

Карусели и посты для соцсетей: кейсы, исследования, анонсы, результаты работы. Параллельно разрабатывались шаблоны документов и email — для удобного обновления командой без дизайнера.

4 AI-видео

AI-сгенерированные видеоролики для крупного ивента в Бразилии: объяснение принципа работы Taara и визуальная демонстрация покрытия сети на больших территориях. Понятный и наглядный формат для аудитории без технического бэкграунда.
